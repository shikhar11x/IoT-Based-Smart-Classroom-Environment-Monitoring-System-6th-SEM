# IoT-Based Smart Classroom Environment Monitoring System

> 📡 An IoT-based minor project for monitoring classroom environmental conditions with real-time cloud visualization.

## Project Overview

This repository contains the complete implementation and documentation of the **IoT-Based Smart Classroom Environment Monitoring System**, developed as a **Minor Project** for the B.Tech (CSE) program at K. R. Mangalam University.
The **IoT-Based Smart Classroom Environment Monitoring System** is designed to continuously monitor and analyze classroom environmental conditions such as **temperature, humidity, and air quality**. A comfortable classroom environment plays a crucial role in enhancing student concentration and learning outcomes. This project contributes to **smart campus initiatives** by providing real-time environmental data through a centralized dashboard.

---

## Problem Statement

Classroom environments significantly affect student focus, health, and academic performance. Traditional classrooms lack real-time monitoring of environmental parameters, leading to discomfort and reduced productivity. There is a need for an **automated, IoT-based monitoring system** that can track these parameters and assist in maintaining an optimal learning environment.

---

## Objectives

* Learn IoT hardware and sensor integration
* Understand real-time data collection and transmission
* Build cloud-connected monitoring systems
* Visualize live sensor data on a dashboard
* Apply basic analytics for smart infrastructure

---

## Proposed Solution

The proposed system uses **IoT sensors connected to a microcontroller** to collect environmental data from classrooms. This data is transmitted to a **cloud-based dashboard**, where it is displayed in real time. Alerts can be generated if conditions exceed predefined thresholds, helping authorities take timely action.
